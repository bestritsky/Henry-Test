## 3. Visual & Interaction Design System

### Aesthetic: "Void & Photon"
- **Backgrounds:** Deepest void black (#030305), not gray.
- **Lighting:** Dynamic "photon" glows that follow the cursor, illuminating content like a flashlight in a dark server room.
- **Typography:**
    - *Headings:* `Space Grotesk` - Technical, futuristic but legible.
    - *Body:* `Inter` or `Geist Sans` - The standard for readability.
    - *Accents:* `JetBrains Mono` - For "code" or "data" elements.
- **Motion:**
    - Heavy use of `Framer Motion`.
    - **Scroll:** Smooth parallax, elements "lock" into place with magnetic precision.
    - **Text:** Words decode/unscramble into view (like a terminal booting up).

## 4. Technical Architecture

### Stack
- **Framework:** **Next.js 15 (App Router)** - For speed and SEO.
- **Styling:** **Tailwind CSS 4** - Variable-based, ultra-lean execution.
- **Animation:** **Framer Motion** - Complex orchestration of layout changes.
- **Deployment:** Vercel (Production), Netlify (Staging/Backup).
- **Clean Code:** TypeScript strict mode, ESLint.

### Component Architecture
1.  **`HypeSwitch`**: Context provider managing global text/theme state.
2.  **`NeuralHero`**: Canvas-based interactive background (using `react-three-fiber` or optimized lightweight shader).
3.  **`ServiceMatrix`**: Grid layout with hover-reveal details.
4.  **`TrustTimeline`**: Vertical scrolling timeline of Henry's career, verified by "data signatures" (icons confirming acquisitions/revenue).
